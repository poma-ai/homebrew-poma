# homebrew-poma

Homebrew tap for [POMA AI CLI](https://github.com/poma-ai/poma-cli) — the CLI for the POMA AI API.

## Install

```bash
brew tap poma-ai/poma
brew install poma
```

## Updating

```bash
brew update
brew upgrade poma
```
